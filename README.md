# road-accident-in-india-
Road Accidents in India: Analysis & Prediction
Web App link:https://roadaccidentpredictor.streamlit.app/

Introduction
🚦 Road Accident Analysis & Severity Prediction Platform

A machine learning–powered analytics platform that predicts road accident severity using multi-year Indian accident datasets. The system enables users to upload CSV files and generate automated insights, visualizations, and model predictions through an interactive web interface.

🎯 Objective

To design a predictive analytics solution that identifies accident severity patterns using supervised machine learning, enabling data-driven road safety planning and proactive risk mitigation.

🖥 Interactive Web Application

Built an interactive dashboard using Streamlit where users can:

Upload custom CSV datasets

Automatically preprocess and clean data

Run supervised ML models

Visualize trends and correlations

View prediction accuracy and performance metrics

Generate graphical insights in real-time

📊 Dataset Coverage

State-wise accident statistics (2016–2020), including:

Total accidents per state

Accidents per lakh population

Accidents per 10,000 vehicles

Injury per lakh population

Injury per 10,000 km of roads

🧹 Data Engineering Pipeline

Missing value imputation

Feature scaling & normalization

Outlier detection

Feature engineering

Categorical encoding

🧠 Machine Learning Models

Implemented and compared multiple supervised algorithms:

Decision Tree

Random Forest

XGBoost

Neural Networks

📈 Evaluation Metrics

Accuracy

Precision

Recall

F1-Score

Best-performing model used for final severity prediction.

Programming: Python
